# nVHosts 0.4.1

This a simple script to manage Virtual Hosts using nGinx and PHP on Linux machines.

### 1) Configuration 

    a) clone de project
    b) cd nVHosts/src
    c) chmod +x nvhosts
    c) sudo ln -s $(pwd)/nvhosts /usr/local/bin/
    d) cd ..
    e) sudo ln -s $(pwd)/templates /usr/local/bin/
